# dust_plugin_derive

Built-in plugin for core derive generation.

## Owns

- `ToString()`
- `Eq()`
- `CopyWith()`
- derive-side shared-analysis collection for cross-file `CopyWith`

## Used by

- `dust_driver`
- `dust_emitter`

- generated derive output changes
- `CopyWith` semantics change
- derive validation or shared-analysis collection changes
