## 0.1.0

- Initial public release of Dust HttpClient annotations.
- Added `HttpClient`, `GenerateTest`, and `DustParseThread`.
- Added HTTP method annotations including `GET`, `POST`, `PUT`, `PATCH`, `DELETE`, `HEAD`, and `OPTIONS`.
- Added request binding annotations including `Path`, `Query`, `Queries`, `Header`, `Headers`, `HeaderMap`, `Body`, `Field`, `Part`, and `Extra`.
- Added request mode and parsing annotations including `FormUrlEncoded`, `MultiPart`, and `HttpParse`.
