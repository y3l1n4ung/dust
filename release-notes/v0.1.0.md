# Dust v0.1.0

First public release of Dust.

You focus on product. We focus on performance.

## Highlights

- Rust-powered Dart code generation for derive, serde, validation, Dio
  HttpClient, routing, state, and DB workflows.
- `dust` CLI with build, check, watch, doctor, and clean commands.
- HttpClient generation backed by real example APIs and generated
  request-mapping tests.
- Product showcase, benchmark-project, and shopping-app fixtures included in
  the repository.

## Included packages

- `dust` CLI from GitHub release artifacts
- Dart packages:
  - `dust_dart`
  - `dust_flutter`
  - `dust_db_sqlite3`

## Notes

- Release binaries are attached to this GitHub release with checksums.
- The release workflow uses this file as the GitHub release notes source.
- Dust Rust crates are not published to crates.io for this release.
