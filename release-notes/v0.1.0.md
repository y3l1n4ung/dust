# Dust v0.1.0

First public release of Dust.

## Highlights

- Rust-powered Dart code generation for derive, serde, and Dio HttpClient
  workflows.
- `dust` CLI with build, check, watch, doctor, and clean commands.
- HttpClient generation backed by real example APIs and generated
  request-mapping tests.
- Product showcase and stress-project fixtures included in the repository.

## Included packages

- Rust crates from `dust_text` through `dust_cli`
- Dart packages:
  - `derive_annotation`
  - `derive_serde_annotation`
  - `dust_http_client_annotation`

## Notes

- Release binaries are attached to this GitHub release with checksums.
- The release workflow uses this file as the GitHub release notes source.
